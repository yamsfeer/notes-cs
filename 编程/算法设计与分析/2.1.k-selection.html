                                          # 选择问题

                                          问题：集合LLL含nnn个不等的实数，求LLL中第iii小的元素。

                                          i=1i = 1i=1，称为最小元素，i=ni = ni=n，称为最大元素，位置处在中间的元素，称为中位数。

                                          • nnn为奇数，中位数唯一，i=(n+1)/2i=(n+1)/2i=(n+1)/2

                                          • nnn为偶数，可指定i=n/2+1i = n / 2 + 1i=n/2+1

                                          # 选最大

                                          算法：遍历nnn个数的数组LLL，令maxmaxmax为当前最大元素，kkk为当前最大元素的下标，当遇到更大的数时，更新maxmaxmax和kkk。

                                          时间：W(n)=n−1W(n) = n - 1W(n)=n−1

                                          // findmax(L)
max = L[1]
for(i from 2 to n)
  if(max < L[i])
    max = L[i]
    k = i
return max, k

                                          # 选最大最小

                                          问题：集合LLL含nnn个不等的实数，求LLL中最大的元素maxmaxmax和最小的元素minminmin。

                                          • 蛮力算法

                                            顺序比较，先选出最大元素maxmaxmax，然后在剩余数组中选出minminmin。

                                            时间：W(n)=(n−1)+(n−2)=2n−3W(n) = (n - 1) + (n - 2) = 2n - 3W(n)=(n−1)+(n−2)=2n−3

                                          • 分组算法

                                            对LLL按两个一组分组，共⌊n/2⌋\lfloor n / 2 \rfloor⌊n/2⌋组，以及可能存在多余的元素kkk，

                                            每组中的较大元素组成MAXMAXMAX，较小元素组成MINMINMIN，

                                            用蛮力算法，在MAXMAXMAX和kkk中选出maxmaxmax，MINMINMIN和kkk中选出minminmin。

                                            举个例子：

                                            时间：W(n)=⌊n/2⌋+2⌈n/2⌉−2=⌈3n/2⌉−2W(n) = \lfloor n / 2 \rfloor + 2\lceil n / 2 \rceil - 2 = \lceil 3n / 2 \rceil - 2W(n)=⌊n/2⌋+2⌈n/2⌉−2=⌈3n/2⌉−2

                                          • 分治算法

                                            将数组从中间划分为两个子数组 L1L_1L1​ 和 L2L_2L2​，

                                            递归地在L1L_1L1​中求max1max_1max1​和min1min_1min1​，递归地在L2L_2L2​中求max2max_2max2​和min2min_2min2​，

                                            那么max=max{max1,max2},min=min{min1,min2}max = max\{max_1, max_2\}, min = min\{min_1,min_2\}max=max{max1​,max2​},min=min{min1​,min2​}。

                                            时间：W(n)=2W(n/2)+2=⋯=3n/2−2W(n) = 2W(n / 2) + 2 = \dots = 3n/2 -2W(n)=2W(n/2)+2=⋯=3n/2−2。

                                          # 小结

                                          选最大：顺序比较，比较次数n−1n - 1n−1

                                          选最大最小（至少3n/2−23n / 2 - 23n/2−2次比较）

                                          • 蛮力算法，比较次数2n−32n - 32n−3
                                          • 分组算法：比较次数⌈3n/2⌉−2\lceil 3n/2 \rceil - 2⌈3n/2⌉−2
                                          • 分治算法：n=2kn = 2^kn=2k，比较次数3n/2−23n/2 - 23n/2−2

                                          选最大算法中蛮力算法已是最优。

                                          选最大最小算法中，分组和分治算法已是最优算法。

                                          # 选第二大

                                          问题：集合LLL含nnn个不等的实数，求LLL中第二大元素secondsecondsecond。

                                          # 蛮力算法

                                          顺序比较找到最大元素maxmaxmax，然后从剩余的n−1n - 1n−1个元素中找最大。

                                          时间：W(n)=n−1+n−2=2n−3W(n) = n - 1 + n - 2 = 2n - 3W(n)=n−1+n−2=2n−3。

                                          # 锦标赛算法

                                          基本思想：能淘汰secondsecondsecond的只有maxmaxmax，记录所有被maxmaxmax淘汰的数，它们中的最大元素即是secondsecondsecond。

                                          1. 两两分组比较，大者进入下一轮，直到只剩下一个元素maxmaxmax为止
                                          2. 在每次比较中淘汰较小元素，将被淘汰元素记录在淘汰它的元素的链表上
                                          3. 检查maxmaxmax的链表，从中找到最大数，即是secondsecondsecond

                                          算法过程：

                                          1. 将nnn个元素两两分组，共⌊n/2⌋\lfloor n/2 \rfloor⌊n/2⌋组
                                          2. 每组的两个数比较，较大数留下，较小数加入较大数链表
                                          3. nnn为奇数，n=⌊k/2⌋+1n = \lfloor k/2\rfloor+1n=⌊k/2⌋+1；nnn为偶数，n=⌊k/2⌋n = \lfloor k/2 \rfloorn=⌊k/2⌋
                                          4. k>1k > 1k>1，继续第2步
                                          5. 剩下的数为maxmaxmax，找到maxmaxmax的链表中的最大值

                                          空间：需要额外空间存放链表，空间换时间

                                          时间：W(n)=n+⌈logn⌉−2W(n) = n + \lceil logn \rceil - 2W(n)=n+⌈logn⌉−2

                                          该算法是所有以元素比较为基本运算的找第二大算法中最好的算法。

                                          # 选中位数的应用

                                          k−selectionk-selectionk−selection的常见应用是选择中位数。

                                          问题：假设有一条主干道和nnn个地点，求主干道的位置，使得每个地点到主干道的长度总和最小。

                                          当主干道位于nnn个地点垂直方向的中位数mmm时，若再将主干道往下或往上移动Δ\DeltaΔ的距离，增加的距离必大于减少的距离。因此，中位数的位置就是解。
