最长公共子序列

给定两个序列，X=x1x2…xm,Y=y1y2…ynX=x_1x_2\dots x_m, Y=y_1y_2\dots y_nX=x1​x2​…xm​,Y=y1​y2​…yn​，求XXX和YYY的最长公共子序列LCSLCSLCS。

举个例子：

蛮力算法：检查XXX的每个子序列是否在YYY中存在，其中XXX有2m2^m2m个子序列，每次检查YYY中是否存在需要O(n)时间。因此蛮力算法最坏情况下时间复杂度为O(n2m)O(n2^m)O(n2m)。

标记函数

每判断一次子序列X,YX,YX,Y的最后一个元素，根据是否相等来决定归约的子问题，用标记函数B[i,j]B[i,j]B[i,j]来表示，B[i,j]B[i,j]B[i,j]的值有三种情况：

  • C[i,j]=C[i−1,j−1]+1⇒ ↖C[i,j] = C[i-1,j-1]+1 \Rightarrow \ \nwarrowC[i,j]=C[i−1,j−1]+1⇒ ↖
  • C[i,j]=C[i,j−1]⇒ ←C[i,j]=C[i,j-1] \Rightarrow \ \leftarrowC[i,j]=C[i,j−1]⇒ ←
  • C[i,j]=C[i−1,j]⇒ ↑C[i,j]=C[i-1,j] \Rightarrow \ \uparrowC[i,j]=C[i−1,j]⇒ ↑

算法伪代码

for i from 1 to m
  C[i, 0] = 0
for i from 1 to n
  C[0, j] = 0

for i from 1 to m
  for j from 1 to n
    if X[i] = Y[j] // 最后一个元素相同
      C[i, j] = C[i-1, j-1] + 1
      B[i, j] = 'left-top'
    else if C[i-1, j] >= C[i, j-1]
      C[i, j] = C[i-1, j]
      B[i, j] = 'top'
    else
      C[i, j] = C[i, j-1]
      B[i, j] = 'left'
