                                                      # Blinn-Phong 模型

                                                      我们能观察到物体，是因为光从物体表面进入了眼睛。而光打到物体上，可以产生三种类型的反射光。

                                                      • 镜面反射高光 ( Specular highlights )
                                                      • 漫反射 ( Diffuse reflection )
                                                      • 环境光 ( Ambient lighting )

                                                      下面我们分别模拟出这三种反射光，然后综合得到具有真实感的光照效果，这就是 Blinn-Phong 模型。

                                                      # shading point

                                                      无论是哪种反射，无论是曲面还是平面，计算光照着色时只考虑在物体表面一个很小的局部平面，称为 shading point，另外也不考虑 shading point 是否在阴影内。

                                                      shading point 和光的关系用以下几个单位向量定义：

                                                      • 平面的法线 n⃗\vec{n}n
                                                      • 光的入射方向 l⃗\vec{l}l
                                                      • 观测方向 v⃗\vec{v}v

                                                      # 漫反射 ( Diffuse Reflection )

                                                      所谓漫反射，就是光线打在物体表面某点时，光线被反射到各个方向。

                                                      # 兰伯特余弦定律

                                                      光是一种能量，光照射到物体时，一部分能量被吸收，剩余的被反射。物体表面的颜色亮度是由它吸收了多少能量决定的。

                                                      兰伯特余弦定律 ( Lambert’s cosine law )：接收到的能量与光照方向和法线方向的夹角余弦成正比。

                                                      这个定律很好理解，垂直照射到物体的部分才是接收到的能量。光与法线的夹角也容易计算。

                                                      cosθ=l⃗⋅n⃗∣l⃗∣∣n⃗∣=l⃗⋅n⃗ cos\theta=\displaystyle\frac{\vec{l}\cdot\vec{n}}{|\vec{l}||\vec{n}|}=\vec{l}\cdot\vec{n} cosθ=∣l∣∣n∣l⋅n​=l⋅n

                                                      # 光衰减

                                                      除了入射角度之外，光源与照射点的距离也应该考虑。直观来说，光离得越远强度越弱。

                                                      # 漫反射公式

                                                      有了光的强度和光与物体平面的夹角，我们就可以计算光的漫反射效果了。

                                                      漫反射的计算公式为：

                                                      Ld=kd(I/r2)max(0,n⃗⋅l⃗) L_d=k_d(I/r^2)max(0,\vec{n}\cdot\vec{l}) Ld​=kd​(I/r2)max(0,n⋅l)

                                                      • III 为点光源光强，I/r2I/r^2I/r2 表示光衰减后到达的强度
                                                      • max(0,cosθ)max(0,cos\theta)max(0,cosθ) 是因为当夹角大于 90 度时，直接忽视。

                                                      其中 kdk_dkd​ 为漫反射系数

                                                      • kd=1k_d=1kd​=1 表示这个点完全不吸收能量
                                                      • kd=0k_d=0kd​=0 表示所有能量都被吸收

                                                      如果把反射系数表示为 rgb 颜色向量，就可以在 shading point上定义一个颜色。

                                                      漫反射光线强度是与出射方向无关的，也就是说从各个方向观察的结果都是一样的。从漫反射的公式也可以看出这一点，公式中没有出现 v⃗\vec{v}v。

                                                      # 高光

                                                      对表面光滑的物体，光的反射效果接近镜面反射，即入射角和出射角相等，比如图中的 RRR。

                                                      当观察方向和镜面反射方向接近时，可以看到高光效果，即 v⃗\vec{v}v 与 RRR 接近。

                                                      # 高光公式

                                                      观察方向与镜面反射方向接近，相当于法线向量与入射方向和观察方向的半程向量 ( 角平分线向量 ) 接近。即

                                                      v⃗=R⟺n⃗=h⃗ \vec{v} = R \quad\Longleftrightarrow\quad \vec{n}=\vec{h} v=R⟺n=h

                                                      于是，和漫反射类似，高光的计算公式为：

                                                      Ls=ks(I/r2)max(0,cosα) L_s=k_s(I/r^2)max(0,cos\alpha) Ls​=ks​(I/r2)max(0,cosα)

                                                      • 之所以引入半程向量是因为半程向量 h⃗\vec{h}h 比 反射反向 RRR 容易计算。Phong 模型就是直接用 RRR 计算，Blinn-Phong 模型相当于做了个改进。
                                                      • 光在接近镜面反射时只有少量能量被吸收，Blinn-Phong 模型是个经验模型，直接忽略了这部分能量，所以没有考虑类似漫反射中的 n⃗⋅l⃗\vec{n}\cdot\vec{l}n⋅l。

                                                      # 指数 p

                                                      半程向量和法线的夹角 α\alphaα 确实可以观察方向和反射方向的接近程度，但它不够精确。

                                                      比如下面第一张图，当观察方向和反射方向成 45 度甚至 80 度时，仍能看到高光，这明显不合理。

                                                      于是我们给 cosαcos\alphacosα 加上若干次余弦，使其对 α\alphaα 变化更加敏感。

                                                      Blinn-Phong 模型中指数的范围 ppp 一般为 100 到 200。

                                                      # 环境光

                                                      前面说过，Blinn-Phong 模型是个经验模型，并不精确。

                                                      Blinn-Phong 模型直接用一个常数表示环境光。

                                                      La=kaIa L_a=k_aI_a La​=ka​Ia​

                                                      环境光比较复杂，精确计算需要用到全局光照的知识。

                                                      综上以上三种模型，得到最终的计算公式：
