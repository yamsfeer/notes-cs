# 序言

本文是 [GAMES101-现代计算机图形学入门](bilibili.com/video/BV1X7411F744) 的笔记。

该教程讲的是计算机图形学的基础，主要包括四部分的内容：

* 光栅化：将三维世界的物体渲染在平面上，这是渲染管线的一个重要组成部分
* 几何：如果表示一个复杂几何体
* 光线追踪：实时光线追踪的主要算法
* 动画仿真：对物理世界规律的仿真

其中渲染管线是基础中的基础，一定要掌握。
